Migration Script Of Todo Lists From Basecamp To Acunote

This script if made for those who want to migrate their todo lists from Basecamp to Acunote.

See project on github

As Acunote does not have (yet) an API, this script is making API calls to get the data from Basecamp and outputs a CSV file that you can import in Acunote.

Usage

Edit the file params.py according to your credentials :

# Basecamp URL for API calls
url = "https://example.basecamphq.com"

# Basecamp API key
apikey = "04f69569a36d306701238466d5ea12cb097b959d"

# Users
# List here all the user IDs in Basecamp
#Â and associate a name in Acunote :
# Basecamp IDs => Acunote names
creatorNames = {
    '7248751' : "quentin",
    '1122334' : "bob",
}

Give the todo list ID in argument to the script and it will output the CSV file to import in Acunote :
